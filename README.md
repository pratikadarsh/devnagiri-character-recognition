# devnagiri-character-recognition
Character recognition for the Devnagiri character dataset.
Dataset can be downloaded from https://web.archive.org/web/20160105230017/http://cvresearchnepal.com/wordpress/dhcd/

---------------------------
        ARCHITECTURE
---------------------------

Model Used: CNN
Layers:     4
FC Layers:  2
Dropout:    Yes(0.4)

---------------------------
        HYPERPARAMETERS
---------------------------

Learning Rate: 1E-3
Epochs:        10
Batch Size:    100

---------------------------
       EVALUATION
---------------------------

Accuracy:       98.53%
Training Loss:  0.02
Eval Loss:      0.04
